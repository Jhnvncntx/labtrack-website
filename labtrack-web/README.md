# LabTrack Website

A modern web application built with **React** and **Vite**, featuring a responsive frontend with optimized development and build performance.

## Tech Stack

- **JavaScript** (56.3%) - Core application logic
- **CSS** (41.2%) - Styling and responsive design
- **HTML** (2.5%) - Markup structure

## Features

- Fast development server with Hot Module Replacement (HMR)
- ESLint configuration for code quality
- Optimized build process with Vite
- Support for multiple React plugins (Oxc and SWC)

## Getting Started

This project uses React with Vite for a minimal, fast setup. For detailed setup instructions and development guidelines, see the project documentation.
